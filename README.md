# JarvisAI

JarvisAI is a Python-based virtual assistant that can perform various tasks and provide assistance in a wide range of areas.
It's designed to help automate tasks, provide information, and enhance your productivity.

## Features

- Voice recognition and synthesis
- Task automation
- Weather forecasts
- and more...

## Getting Started

These instructions will help you get a copy of the project up and running on your local machine.

### Prerequisites

- Python (version 3.4)
- speech_recognition 
- os
- win32com.client
- webbrowser
- openai
- datetime

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Cyphersk21/JarvisAI.git
   ```

2. Navigate to the project directory:

   ```bash
   cd JarvisAI
   ```

### Usage

To start JarvisAI, run the following command:

```bash
python JarivisAI.py
```

Follow the on-screen instructions to interact with JarvisAI.

## Configuration

- You can customize JarvisAI's behavior by editing the `config.py` file.
